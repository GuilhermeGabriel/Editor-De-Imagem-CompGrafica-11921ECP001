## Instalação

```bash
git clone https://github.com/GuilhermeGabriel/Editor-De-Imagem-CompGrafica-11921ECP001.git
cd Editor-De-Imagem-CompGrafica-11921ECP001
npm install
```

## Executando, abra http://localhost:3000 para ver no seu navegador.

```bash
npm run start
```
